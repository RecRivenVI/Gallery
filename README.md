# Gallery

> 开发状态：架构与工程规范已收敛，正式产品代码尚未建立；下一步是阶段 0 契约骨架和 Walking Skeleton。

Gallery（画廊，代码代号 `gallery`）是面向个人及可信局域网的本地优先、只读媒体目录产品。它计划以 Go、SQLite 和 Web/PWA 构建独立后端与多客户端体验，不兼容或迁移任何旧 Gallery 实现。

当前仓库包含：

- [工程文档与 ADR](Documents/README.md)；
- [v1 实施计划](Documents/指南/01-v1实施计划.md)；
- [验证记录](Documents/证据/验证记录.md)；
- `Test-Bench/cleanroom-lab*` 早期净室原型、样本和结果。

开始开发前请先阅读 [AGENTS.md](AGENTS.md)。当前没有可供普通用户安装的正式版本。
